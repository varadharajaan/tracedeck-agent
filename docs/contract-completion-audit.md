# Contract Completion Audit

The contract completion audit is generated by:

```powershell
python ./devctl.py audit
```

It writes:

```text
data/local/output/contract-completion-audit.json
data/local/output/contract-completion-audit.txt
```

## Remaining or partial work

TraceDeck should not be described as fully end-to-end complete until the audit
has no missing or attention items. The audit intentionally separates working
runtime proof from remaining packaging, provider, and production-readiness gaps.

Use this before product demos, PRs, and handoffs.
