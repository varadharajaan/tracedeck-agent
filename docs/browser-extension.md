# Browser Extension

The browser extension skeleton lives under:

```text
browser-extension/
```

The intended browser bridge is domain/category metadata only for Chromium-based
browsers such as Chrome, Edge, and Brave. It must not collect raw URLs, page
titles, cookies, tokens, passwords, page contents, screenshots, or private data.

Local browser activity page:

```text
http://127.0.0.1:18080/browser-activity
```
